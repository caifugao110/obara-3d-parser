# 用户指南

本文面向普通使用者，介绍从模型导入到查看仿真结果的完整流程。

## 主界面组成

- **3D 视图**：显示模型、网格、结果云图，并用于拾取面或零件。支持多种显示模式和交互操作。
- **材料库面板**：按分类列出材料，支持为不同零件分配材料，可加载外部材质库文件夹。
- **分析设置面板**：显示零件、网格、约束、载荷、坐标系和求解器设置。
- **结果面板**：显示最大位移、最大 von Mises 应力、安全系数和载荷面位移报告。
- **状态栏**：提示当前拾取模式、网格化状态、求解状态和错误信息。
- **工具栏**：提供快捷操作按钮，包括导入、视图切换、颜色设置、探测模式等。

## 典型分析流程

### 1. 导入模型

点击"文件 > 导入 STEP 数模..."导入 STEP 文件。程序会通过 Gmsh/OpenCASCADE 读取几何体，并为显示和拾取生成表面三角网格。

也可以点击"文件 > 生成测试梁"创建内置测试梁，用于快速熟悉流程。

**多零件模型支持**：
- 如果 STEP 文件包含多个 3D 实体，每个实体会作为独立零件导入。
- 每个零件会自动编号（如 `Part_1`、`Part_2`）。
- 多零件模型导入后，可通过点击 3D 视图中的零件进行选择。

建议：

- STEP 模型应为实体而非仅曲面。
- 模型尺寸单位按毫米理解。
- 导入后先旋转、缩放视图，确认几何完整。
- 多零件模型导入后，状态栏会提示零件数量。

### 2. 分配材料

#### 2.1 为零件分配材料

1. 在材料库面板的"选择零件"下拉框中选择目标零件（多零件模型时）。
2. 在材料分类树中展开分类，选择所需材料。
3. 查看材料属性预览（弹性模量、泊松比、密度、屈服强度等）。
4. 点击"应用材质到零件"完成分配。

所有参与求解的零件都需要分配材料，否则分析无法运行。

#### 2.2 加载外部材料库

程序支持从外部文件夹加载自定义材质库：

1. 点击"文件 > 选择材质库文件夹..."菜单。
2. 在弹出的对话框中选择包含材质JSON文件的文件夹。
3. 程序会自动扫描该文件夹下所有 `*.json` 文件，并合并材料数据。
4. 如果当前使用的材料在新库中不存在，会提示重新选择。

**注意**：外部材质库的JSON格式需与内置的 `sldmaterials.json` 兼容。

### 3. 生成网格

1. 在分析设置面板中通过滑块调整网格尺寸。
2. 点击"网格划分"按钮。
3. 等待网格化完成，状态栏会显示网格化进度和结果。

网格尺寸决定了计算精度和速度：

- 较小的网格尺寸：更高的精度，但计算时间更长。
- 较大的网格尺寸：更快的计算速度，但精度较低。

建议先使用较大的网格尺寸进行快速验证，确认模型和约束设置正确后再细化网格。

### 4. 添加固定约束

1. 点击"拾取固定面"按钮。
2. 在 3D 视图中点击需要固定的面（可多选）。
3. 选中的面会高亮显示，并在分析设置面板中列出。
4. 使用"删除"按钮移除不需要的约束面。

固定约束意味着该面上的所有节点位移为零。

### 5. 添加载荷

1. 点击"拾取加压面"。
2. 在 3D 视图中点击受载面。
3. 在弹窗中输入载荷大小，并选择载荷方向模式。

支持两类载荷：

- **按面法向压力**：压力沿选中面的法向分布。
- **指定方向力/压力**：按输入方向向量施加载荷，需指定 X/Y/Z 分量。

载荷会在 3D 视图中以箭头形式显示，箭头方向表示载荷方向。

**编辑载荷**：双击载荷列表中的载荷项，可重新设置载荷参数并选择新的加载面。

单位约定见"单位约定"。

### 6. 设置分析坐标系

坐标系用于定义输出位移的方向。

- 全局坐标系：使用默认的 XYZ 坐标系。
- 用户自定义坐标系：通过三点定义新的坐标系。

### 7. 选择求解器

在分析设置面板的"求解器"下拉框中选择：

- **内置求解器**：Python/SciPy 实现的 Tet4 线弹性求解器，无需外部依赖。
- **CalculiX**：导出 CalculiX/Abaqus 格式输入文件并调用外部求解器。

使用 CalculiX 需要提前安装并配置环境变量。

### 8. 运行仿真

点击"运行仿真"按钮。

程序会：

1. 检查所有必要条件是否满足（材料、网格、约束、载荷）。
2. 在后台线程中运行求解器，不阻塞界面。
3. 显示求解进度。
4. 求解完成后自动显示结果。

### 9. 查看结果

仿真完成后，程序会自动显示位移 ISO 图。

**结果面板内容**：

- 最大位移值（按坐标系分解）。
- 最大 von Mises 应力。
- 安全系数（基于最小屈服强度）。
- 载荷面位移报告。

**视图切换**：

- 原始视图：显示未变形的模型。
- 网格视图：显示四面体网格。
- 位移图：显示位移 ISO 云图。
- 应力图：显示 von Mises 应力 ISO 云图。

## 视图与拾取建议

- 拾取面前先旋转到能清晰看到目标面的角度。
- 对复杂模型可先显示网格，确认面分组是否符合预期。
- 如果拾取错误，可从约束或载荷列表中删除后重新拾取。
- 使用重置布局功能可恢复默认面板位置。
- 多零件模型中，可通过点击 3D 视图切换当前选中的零件。

## 颜色设置

### 面颜色标记

1. 点击工具栏中的"面颜色"按钮，选择所需颜色。
2. 点击"应用面颜色"按钮进入面拾取模式。
3. 在 3D 视图中点击目标面，该面会被标记为选中的颜色。
4. 使用"清除所有面颜色"按钮移除所有面颜色标记。

### 零件颜色标记

1. 点击工具栏中的"零件颜色"按钮，选择所需颜色。
2. 点击"应用零件颜色"按钮进入零件拾取模式（多零件模型）。
3. 在 3D 视图中点击目标零件，该零件会被标记为选中的颜色。
4. 如果只有一个零件，会直接应用颜色。
5. 使用"清除所有零件颜色"按钮移除所有零件颜色标记。

## 探测模式

仿真完成后，可使用探测模式查看任意点的详细结果：

1. 点击工具栏中的"探测模式"按钮。
2. 在 3D 视图中点击模型上的任意位置。
3. 状态栏会显示该点的坐标、位移分量（UX/UY/UZ）、合成位移和 von Mises 应力。
4. 再次点击"探测模式"按钮退出探测模式。

**注意**：探测模式只能在仿真完成后使用。

## 结果解释注意事项

### 安全系数计算

安全系数基于 von Mises 应力和材料屈服强度：

```
安全系数 = 最小屈服强度 / 最大 von Mises 应力
```

对于多材料模型，使用所有零件中最小的屈服强度。

### 单位约定

程序使用以下单位约定：

- 长度：毫米（mm）
- 力：牛顿（N）
- 压力：兆帕（MPa）
- 应力：兆帕（MPa）
- 弹性模量：吉帕（GPa）

### 位移放大

为了便于观察，位移云图会将实际位移放大显示。放大比例会在图例中注明。

### 线性分析假设

当前求解为线弹性静力分析，假设：

- 材料为线弹性（应力与应变成正比）。
- 变形很小（小变形假设）。
- 无塑性变形。
- 无动态效应。

如果实际工况超出这些假设范围，分析结果可能不准确。

## 导出说明

当前主窗口没有提供手动导出结果图片或手动导出 `.inp` 文件的按钮。

选择 CalculiX 求解器运行仿真时，程序会在后台自动生成 CalculiX/Abaqus 输入文件并调用外部 `ccx`。如需保留或自定义导出文件，请参考 `app/solver_backends.py` 中的 CalculiX 后端实现。

## 常见问题

### 为什么有些面无法拾取？

- 可能是视角问题，尝试旋转到不同角度。
- 确保在正确的拾取模式下（固定面或载荷面）。
- 网格未生成时无法拾取面。

### 为什么安全系数显示为无穷大？

如果最大应力小于材料的屈服强度，安全系数会显示为无穷大，表示结构处于弹性范围内。

### 如何重置所有设置？

当前主窗口没有"新建"菜单。可使用"视图 > 清空面板"清除约束、载荷、坐标系和结果；如需完全重新开始，请重新导入模型或关闭并重新打开程序。
